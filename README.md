# gipermiumbrowser
I made my own simple browser from scratch using Python.

# How i make this
I downloaded webkit2gtk also used PyQt5 PyQtWebEngine so if your browser doesn't work download them.

On Arch Linux:

``sudo pacman -S webkit2gtk``

On Debian 12:

``sudo apt install webkit2gtk``

PyQt5 and PyQtWebEngine are downloaded using pip:

``pip install PyQt5 PyQtWebEngine``
